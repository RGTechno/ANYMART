# ANYMART
Your All Time Shop !

**ANYMART is a multi-vendor delivery platform, where multiple vendor can register their store and make their business online.**

**TECH STACK**
- Flutter
- Firebase Auth
- Firebase Storage
- Firebase Firestore Database
